## How to run -

- I have provided WebBuild and WindowsBuild for this project

- To run the WindowsBuild download this repo and open WindowsBuild folder , then double click the "Digital Twin" application to run the build.

- To run the WebBuild, open this link - https://play.unity.com/en/games/342c75c9-15e3-48a2-a606-a43ea216108d/webbuild and press Play to run he build.
